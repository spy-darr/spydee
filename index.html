// ⚠ ACTION REQUIRED — CANNOT CONNECT TO AI FROM file://
Browsers block API calls when a page is opened directly as a file. This is a browser security rule called CORS. Spydee needs to be served over HTTP to talk to Groq's AI.

One-time fix — choose the easiest option for you:
  1. VS Code: Install the Live Server extension → right-click spydee.html → "Open with Live Server". Done. Use the http://127.0.0.1:5500 URL it opens.
  2. Python (any OS): Open a terminal in the folder and run python -m http.server 8080 then open http://localhost:8080/spydee.html
  3. Node.js: Run npx serve . in the folder, then open the URL it shows.
  4. Android / iOS: Upload the file to any free static host (Netlify Drop, GitHub Pages) and open that URL — it will work perfectly on mobile too.
